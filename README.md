# Cars

## Run the API
> Run: $ docker-compose up --build -V

> you can reach the api via the link [API](http://localhost:3000/cars-details/)

## Run the website
> make sure the api is running

> Run: $ cd cars_website

> Run:  $ npm install

> Run: ~~ $ npm start 

> you can reach the api via the link [website](http://localhost:4200)
